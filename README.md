
# Flask Expense Tracker

A web-based expense tracking application built with Flask and SQLAlchemy that helps users manage and organize their daily expenses.

## Features

- Add expenses
- View all expenses
- Edit expenses
- Delete expenses
- Filter expenses by category
- Calculate total expenses
- Store data using SQLite database
- Clean and simple user interface

## Technologies Used

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
